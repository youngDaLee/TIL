목표

- 메서드가 어떻게 호출되고 메모리에 어떤 변화를 일으키는지

## 질문

## 정리
JDK > JRE > JVM
* JDK - 자바 개발 도구 : 자바 소스 컴파일러(javac.exe)포함
* JRE - 자바 실행 환경 : 자바 프로그램 실행기(java.exe)포함
* JVM - 자바 가상 기계 : 각 플랫폼에 맞는 JVM이 프로그램을 구동하도록 해줌

자바 메모리
- 코드 실행 영역
- 데이터 저장 영역(static영역)
  - 스택(stack)영역
  - 힙(Heap)영역
