# BOJ
[계단 수](https://www.acmicpc.net/problem/1562)

### 문제 이해하기
- 45656은 인접한 모든 자리의 차이가 1 -> 이런게 계단 수
- N이 주어질 때 길이가 N이면서 0부터 9까지의 숫자가 모두 등장하는 계단 수가 총 몇개인지 구하는 프로그램
- 0으로 시작하는수는 계단수가 아님

### 문제 접근 방법
비트마스킹과 DP를 이용해 풀이
- 모두 같은데 2진수 마지막 수만 다르면 계단수임
- AND로 했을 때 짝수인 쪽과 같으면 계단수
