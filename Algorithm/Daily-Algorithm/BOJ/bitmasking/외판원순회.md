# 프로그래머스
[외판원순회](https://www.acmicpc.net/problem/2098)

### 문제 이해하기
- 한 도시에서 출발해서 N개의 도시를 거쳐 다시 원래의 도시로 돌아옴(외판원)
- 한 번 갔던 도시로는 다시 갈 수 없음
- W[i][j] 형태로 도시 i 에서 도시 j로 가는 비용이 주어짐
- W[i][j]가 0이면 i -> j 로 못간다는 뜻
- 가장 적은 비용을 들이는 외판원 순회 비용
